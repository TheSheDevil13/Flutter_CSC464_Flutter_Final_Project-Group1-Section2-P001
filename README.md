# Flutter Final Project - CSC464

Welcome to our group project! This is a Flutter application initialization for Group 1, Section 2.

## Collaboration Guide

To get started with this project, follow these steps:

### 1. Prerequisite
Ensure you have Flutter installed on your machine. You can check this by running:
```bash
flutter --version
```

### 2. Clone the Repository
If you haven't already, clone the repository to your local machine:
```bash
git clone https://github.com/TheSheDevil13/Flutter_CSC464_Flutter_Final_Project-Group1-Section2-P001.git
```

### 3. Setup Project
Navigate to the project directory and get the dependencies:
```bash
cd Flutter_CSC464_Flutter_Final_Project-Group1-Section2-P001
flutter pub get
```

### 4. Running the App
To run the app on your emulator or connected device:
```bash
flutter run
```
